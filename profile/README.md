# GGX Project - Open source Rust XeBuild replacement 

A collection of ulities ported from othet projects to form a xeBuild replacement

## Quickstart
Download ggx-cli.exe from releases. GGX-CLI documentstion is available [here]()

## Developer
GGX-CLI acts as a reference implementation of LibGGX. All commands exposed in cargo are also available in FFI. GxInterface documentation available [here]()

## Components

- GGX-CLI - Official CLI and Reference implementation
- GxInterface - LibGGX interface / endpoint
- GxBuild - NAND and FlashFS manager
- GxPatcher - Multi format NAND patcher
- GxCrypt - Xbox 360 Cryptographic Keyring
- GxSTFS - STFS Explorer, Extractor and Verify
